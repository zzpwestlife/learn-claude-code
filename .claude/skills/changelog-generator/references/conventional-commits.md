# Conventional Commits Quick Reference

## Common Types
- feat: new feature
- fix: bug fix
- docs: documentation only
- refactor: code change without behavior change
- perf: performance improvement
- test: add or fix tests
- chore: tooling or maintenance

## Changelog Mapping
- feat: Added
- fix: Fixed
- docs: Documentation
- refactor: Changed
- perf: Improved
- test: Tests
- chore: Maintenance
