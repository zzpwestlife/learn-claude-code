# IELTS Rubrics Reference

Condensed rubrics for the IELTS Academic tasks this skill supports. Based on British Council/IDP published scoring guides. Use these to grade the user's practice output in the sub-skills.

**General note**: IELTS uses human examiners in the real exam. Claude's scoring is a human approximation — it correlates with, but is not identical to, the official band descriptors. Treat scores as directional, not exact.

---

## IELTS Band Score Overview (9-point scale)

| Band | Skill Level | Description |
| :---: | :--- | :--- |
| 9 | Expert | Full command of the language |
| 8 | Very Good | Fully operational command |
| 7 | Good | Operational command, minor inaccuracies |
| **6** | **Competent** | **Effective command, some inaccuracies** |
| 5 | Modest | Partial command, copes with overall meaning |
| 4 | Limited | Basic competence in familiar situations |
| 3 | Extremely Limited | Conveys only general meaning |
| 2 | Intermittent | Very limited communication |
| 1 | Non-user | No ability to use the language |

**Target zone**: Most academic institutions require 6.5 or 7.0 overall.

## Task 2: Essay Writing

**Prompt format**: An argumentative/discursive topic, 250-word response in 40 minutes.

**Four scoring criteria** (each 0-9, averaged):

### 1. Task Achievement / Response (TA)

| Band | Descriptor |
| :---: | :--- |
| 9 | Fully addresses all parts of the task; presents a fully developed position |
| 7 | Addresses all parts of the task; presents a clear position; presents, extends, supports main ideas |
| 6 | Addresses all parts of the task; presents a relevant position; presents some ideas |
| 5 | Addresses the task only partially; position may be unclear |
| 4 | Responds to the task only in a minimal way |

**Key requirements**:
- Answer all parts of the question (if it has multiple parts)
- Take a clear position (agree/disagree/balanced)
- Develop ideas with reasons, examples, or evidence
- Stay on topic

### 2. Coherence and Cohesion (CC)

| Band | Descriptor |
| :---: | :--- |
| 9 | Uses cohesion in such a fully fluid way; paraphrasing is seamless |
| 7 | Sequences information logically; manages cohesion effectively |
| 6 | Logically organizes information; uses a range of cohesive devices (although some may be faulty) |
| 5 | Presents information with some organization; uses some cohesive devices (may be over/underused) |

**Key requirements**:
- Clear overall structure: Introduction → Body Paragraphs → Conclusion
- Use cohesive devices: Firstly, Moreover, However, In conclusion, etc.
- Don't overuse the same connector
- Paragraphing: one main idea per paragraph

### 3. Lexical Resource (LR)

| Band | Descriptor |
| :---: | :--- |
| 9 | Full flexibility and precision; rare minor errors |
| 7 | Wide vocabulary range; occasional errors in word choice/spelling |
| 6 | Adequate vocabulary; attempts to use less common words; some errors in spelling/word formation |
| 5 | Limited vocabulary; frequent errors in word choice/spelling |

**Key requirements**:
- Use topic-appropriate vocabulary (not just everyday words)
- Show range: don't repeat the same words
- Collocations matter: "make a decision" not "do a decision"
- Spelling must be accurate

### 4. Grammatical Range and Accuracy (GRA)

| Band | Descriptor |
| :---: | :--- |
| 9 | Full flexibility and precision; rare minor errors |
| 7 | Uses a variety of complex structures; good control; few errors |
| 6 | Uses mix of simple and complex sentences; reasonable control; some errors |
| 5 | Uses only limited structures; errors may cause difficulty |

**Key requirements**:
- Mix of sentence types: simple, compound, complex
- Complex structures: relative clauses, conditionals, passive voice, subjunctive
- Punctuation: commas, periods, semicolons
- Subject-verb agreement must be correct

**How to grade in the sub-skill**:
1. Read the essay
2. For each criterion, assign a band score (1-9) and state the evidence
3. Calculate overall: (TA + CC + LR + GRA) / 4
4. Report band score + dimension breakdown + the ONE dimension that would give the biggest score improvement next time

## Task 1: Graph/Letter Writing

**Prompt format**: Describe visual information (graph, chart, diagram) or write a formal/informal letter, 150+ words in 20 minutes.

**Four scoring criteria** (each 0-9, averaged):

### 1. Task Achievement (TA)

| Band | Descriptor |
| :---: | :--- |
| 9 | Fully satisfies all requirements; presents fully developed response |
| 7 | Covers the requirements; presents a clear overview; data is accurately reported |
| 6 | Covers the requirements; key features are identified; data is adequately reported |
| 5 | Partially covers requirements; some features may be inadequately covered |

**Key requirements (Graphs)**:
- Select and report the main features
- Describe trends (upward, downward, stable)
- Identify key figures: highest, lowest, starting, ending
- Make comparisons where relevant
- Do NOT give reasons or opinions (that's Task 2)

**Key requirements (Letters)**:
- Use appropriate tone (formal/informal/semi-formal)
- Cover all bullet points in the prompt
- Write in paragraph form

### 2. Coherence and Cohesion (CC)

Same as Task 2 — clear structure, cohesive devices, paragraphing.

### 3. Lexical Resource (LR)

Same principles as Task 2, but vocabulary is more specialized:
- **Graph vocabulary**: increased significantly, peaked at, declined, remained stable, fluctuated, reached a plateau
- **Trend language**: there was a sharp rise, a gradual decrease, an upward trend, a dramatic fall

### 4. Grammatical Range and Accuracy (GRA)

Same principles as Task 2, but:
- Past tense is often required (describing what happened)
- Passive voice is common (was recorded, was observed)
- Comparative/superlative forms (the highest point, the most significant increase)

**How to grade in the sub-skill**:
1. Read the response
2. Check word count (150 minimum)
3. For each criterion, assign a band score (1-9) and state the evidence
4. Calculate overall: (TA + CC + LR + GRA) / 4
5. Report band score + dimension breakdown

## Speaking Part 2: Long Turn (Cue Card)

**Prompt format**: Talk for 1-2 minutes on a topic based on a cue card (4 bullet points). 1 minute preparation.

**Four scoring criteria** (each 0-9, averaged):

### 1. Fluency and Coherence (FC)

| Band | Descriptor |
| :---: | :--- |
| 9 | Speech is fully fluent; can account for and incorporate each point naturally |
| 7 | Is willing to speak at length; uses discourse markers and connectors; some hesitation but rarely repeats |
| 6 | Is willing to speak at length; uses cohesive devices but not always appropriately; occasional repetition |
| 5 | Usually maintains flow but may repeat or self-correct; limited use of cohesive devices |

**Key requirements**:
- Speak continuously for 1-2 minutes
- Use linking words naturally (Well, Actually, I mean)
- Don't pause too often or for too long
- Cover all bullet points on the cue card

### 2. Lexical Resource (LR)

Same principles as Writing, but vocabulary is spoken:
- Avoid reading from notes
- Use natural spoken phrases
- Circumlocution when you don't know a word

### 3. Grammatical Range and Accuracy (GRA)

Same principles as Writing, but more tolerance for spoken structures:
- Spoken grammar is less formal
- Contracted forms are natural (I'm, didn't, can't)
- Minor errors in complex sentences are more forgivable

### 4. Pronunciation (P)

| Band | Descriptor |
| :---: | :--- |
| 9 | Full range of pronunciation features used; all sounds clear |
| 7 | Shows all pronunciation features; mostly clear; a few minor issues |
| 6 | Uses a range of pronunciation features; mostly clear; some errors |
| 5 | Can generally be understood; some features may cause difficulty |

**Key requirements**:
- Speak clearly enough to be understood
- Word stress should be correct
- Intonation should be natural
- The user self-reports this criterion

**How this sub-skill grades**:

1. The user describes the topic in text (pasting what he would say)
2. Claude scores **FC**, **LR**, **GRA** from the transcript
3. For **Pronunciation**, Claude asks the user to self-rate 1-9 honestly and records the answer
4. Output: band score + dimension breakdown + the ONE structural element he missed

## Speaking Part 3: Discussion

**Prompt format**: Discussion with examiner about abstract ideas related to Part 2 topic. 4-5 minutes.

**Same four criteria as Part 2**, but expects:
- More complex vocabulary and ideas
- Ability to speculate and hypothesize
- Comparative structures (In some cultures... whereas in others...)
- Examples and explanations for opinions

## Read Aloud (Speaking Warm-up)

**Purpose**: Warm-up practice for pronunciation and oral fluency.

**Prompt format**: A 50-70 word passage. 30 seconds to prepare, then read aloud.

**How this sub-skill works**:

1. Claude provides a difficulty-calibrated passage
2. The user reads it aloud (ideally recording himself)
3. The user self-reports: fluency (1-9), pronunciation (1-9), any words tripped on
4. Claude flags high-risk words in the passage (long syllables, unusual stress) and asks the user to confirm each one sounded correct

**Difficulty progression**:
- **Week 1-2**: Concrete topics, common vocabulary, short sentences
- **Week 3-4**: Abstract topics (economics, science), mixed vocabulary, compound sentences
- **Week 5+**: Academic register, complex sentence structures matching IELTS Reading passages

## IELTS Overall Band Calculation

**Writing and Speaking**: Average of all criteria.

**Overall**: Average of four papers (Listening, Reading, Writing, Speaking), rounded to nearest whole or half band.

| Component | Weight |
| :--- | :---: |
| Listening | 25% |
| Reading | 25% |
| Writing | 25% |
| Speaking | 25% |

**Typical targets**:
- **6.0**: Minimum for many universities
- **6.5**: Required by most for postgraduate programs
- **7.0**: Competitive for top universities
- **7.5+**: Elite programs

These are internal approximations — use them to set weekly targets, not to promise exam scores.
